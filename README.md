# Classify Questions
Trains and tests classification models on the QANTA question dataset.

## Usage
1. ```pip install -r requirements.txt```
2. ```python setup.py```
3. ```python run.py [classifiers to train and test]```
    ### Models included
    * BERT (`bert`)
    * DistilBERT (`distil-bert`)
    * ConvBERT (`conv-bert`)
    * ELECTRA (`electra`)
